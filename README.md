# Password Javascript Game

Attempt to guess the word in 10 guesses or less.

## How to Play

Computer will Provide hint and pick the work.

The word is not case sensitive.


## Rails model

- Game

- Words
  - World __have many__ hints (at leat 10)

- Hints
